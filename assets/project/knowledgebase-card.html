📚 KnowledgeBase

Your team's knowledge, on your own server.دانش تیم شما، روی سرور خودتان.

A self-hosted knowledge management system for internal documentation — with an article dashboard, intern tracking, and a task manager with unlimited nested subtasks and rich filtering. Fully independent of cloud services: all data persists in local JSON files via lowdb, and the whole app ships as a single Docker container you can run on any machine.

یک سیستم مدیریت دانش سلف-هاست برای مستندات داخلی — با داشبورد مقاله‌ها، پیگیری کارآموزها و مدیریت وظایف با زیروظیفه‌های تودرتوی نامحدود و فیلترهای متنوع. کاملاً مستقل از سرویس‌های ابری: همه داده‌ها در فایل‌های JSON محلی با lowdb ذخیره می‌شود و کل اپ در قالب یک کانتینر Docker روی هر ماشینی اجرا می‌شود.
